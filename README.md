# Testing

This repository is a throwaway test bed used to exercise **Loop watcher** experiments. It exists solely to validate the Loop watcher's issue-to-PR automation flow (e.g. opening issues, having an agent pick them up, and producing pull requests).

There is no real product, application, or service here — the contents of this repo are disposable and may be reset or repurposed at any time.

## Scope

- Used for Loop watcher / issue-to-PR flow testing only.
- Safe to experiment in: no production systems, external integrations, or real user data are involved.
- Files and history may be cleared or rewritten as part of ongoing tests.

## Out of scope

- Not a real project — do not treat any code, config, or docs here as production-ready.
- No automation beyond what's needed to test the Loop watcher flow.
